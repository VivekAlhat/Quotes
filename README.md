![npm](https://img.shields.io/npm/v/@vivekalhat/randomquotes)

Simple NPM Package To Generate Random Quotes

## Installation

<code>npm i @vivekalhat/randomquotes</code>

## Usage

```js
const qt = require('@vivekalhat/randomquotes');
var quote = qt.getQuote();
console.log(quote);
```
